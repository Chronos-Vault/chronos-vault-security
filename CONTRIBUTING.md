## üîç Bug Hunting

### What We're Looking For

| Severity | Examples | Tier Impact |
|----------|----------|-------------|
| Critical | Consensus bypass, fund theft | Guardian+ |
| High | Validator manipulation, DoS | Researcher+ |
| Medium | Edge case bugs, logic errors | Researcher |
| Low | Minor issues, optimizations | Watcher+ |

### Scope

**In Scope:**
- Smart Contracts (Solidity, Rust, FunC)
- Formal Proofs (Lean 4)
- ZK Circuits (Circom)
- Cross-chain messaging

**Out of Scope:**
- Third-party dependencies
- Already known issues
- Theoretical attacks without PoC

## üìù How to Contribute

### 1. Report a Bug
```
1. Check existing issues first
2. Create detailed issue with:
   - Title: [SEVERITY] Brief description
   - Steps to reproduce
   - Expected vs actual behavior
   - Proof of concept (if applicable)
3. Wait for triage (24-48 hours)
```

### 2. Submit Code
```
1. Fork the repository
2. Create feature branch
3. Make changes with tests
4. Submit pull request
5. Respond to review feedback
```

### 3. Improve Documentation
- Fix typos or unclear explanations
- Add examples or tutorials
- Translate to other languages
